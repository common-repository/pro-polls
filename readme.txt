== Description ==
Polling plugin that allows you to create polls that have multiple questions and allow the members of your site to vote on them.

Its super simple to use as you don't need account on any other polling site. It saves all the polls and the results on your website.

You can place multiple polls on same page if needed.

Main features:
 
- Allow the admin to post a multi-question poll.
- Only allow users that are LOGGED IN to vote.
- Only allow each logged-in member to vote only once on each question.
- Compiles the results so the admin could easily view the total votes.


== Installation ==
Install Pro Polls plugin.
Activate the plugin.
Thats it!

== Frequently Asked Questions ==
How do I add polls on my pages?
Simple. Just use shortcodes provided along each polls on the list.

How do I view results on the polls?
Just go to the results page for each pool. It will show you the submissions in nice format for each questions and also show you what members have already submitted the poll and who hasn\'t.
